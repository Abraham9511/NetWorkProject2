#### 分析
##### 生成特定的拓扑结构
由于每台电脑一般只有一个网卡，不能做到连接不同的网段。
为了仍然能模拟出物理链路的拓扑结构，每台电脑维护一张记录可以连接的电脑的ip的表格（这是ip层的通讯），该电脑只会认为在表格中的这些电脑的ip是直接相连的



不同的电脑之间交换路由信息
选举总的路由器
生成链路状态信息汇总到总的路由机
总的路由器广播
各个路由器接受到信息并且更新路由表
发送消息
若是链路断开，那么邻近的链路会刷新信息
